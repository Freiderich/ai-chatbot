# AI Chatbot - Python & Flask

# Description
A pixel-style chatbot web application with a modern chat interface.
Built with Python (Flask), HTML, CSS, and JavaScript.
Supports dynamic responses via a rule-based backend.
Can be extended to integrate AI models like OpenAI GPT for intelligent conversations.

# Features
- Pixel-inspired UI with responsive design
- Rule-based dynamic chat responses
- Clean and interactive frontend
- Easily extendable to AI/NLP models
- Minimal setup required

# Setup & Usage
1. Activate virtual environment:
   $ python -m venv venv
   $ venv\Scripts\activate       # Windows
   $ source venv/bin/activate    # macOS/Linux

2. Install dependencies:
   $ pip install -r requirements.txt

3. Run Flask backend:
   $ cd backend
   $ python main.py

4. Open your browser at:
   http://127.0.0.1:5000/
